### Как загружать на сервер:
1. Локально: `git add <файлы> && git commit -m "message" && git push origin main`
2. На сервере SSH:
   ```bash
   cd /opt/glass-bot
   git pull origin main
   sudo systemctl restart glass-bot
   sudo systemctl status glass-bot
   ```
3. Если Conflict ошибка: `sudo systemctl stop glass-bot` → удалить PID файл если остался → `sudo systemctl start glass-bot`
4. Проверить процессы: `ps aux | grep "glass-bot"` (НЕ трогать agrio-bot и другие!)
5. Включить автозапуск: `sudo systemctl enable glass-bot`
