# 🔍 Telegram Bot для подбора стёкол

Бот помогает подобрать защитное стекло для телефона по модели.

## ✨ Возможности

- 🔍 **Умный поиск** — находит даже с опечатками (нечёткий поиск)
- 📱 **Inline-кнопки брендов** — не нужно писать название, просто выбери
- 📊 **Статистика** — сколько пользователей, поисков, активность
- 📜 **История поисков** — последние 5 запросов пользователя
- 🔥 **Популярные запросы** — топ-10 за неделю
- 📩 **Рассылка** — отправка сообщения всем пользователям
- 👥 **Управление пользователями** — блокировка/разблокировка
- 💾 **Автобэкапы** — базы данных и compatibility.json
- 🌍 **Мультиязычность** — русский, английский, арабский
- 🔗 **Партнёрские ссылки** — монетизация
- 📝 **Логирование** — все ошибки и действия записываются
- 🔧 **Админ-панель** — полная управление ботом

## 📁 Структура проекта

```
├── main.py                    # Точка входа (запуск бота)
├── config.py                  # Настройки, тексты, кнопки
├── .env                       # Секреты (не в Git)
├── compatibility.json         # База стёкол
├── bot.db                     # SQLite база (создаётся автоматически)
├── requirements.txt
│
├── handlers/
│   ├── start.py               # /start, /feedback
│   ├── search.py              # Поиск, inline-кнопки, история
│   └── admin.py               # Полная админ-панель
│
├── database/
│   └── __init__.py            # SQLite БД (пользователи, поиски, настройки)
│
└── utils/
    ├── search.py              # Нечёткий поиск + исправление опечаток
    ├── logger.py              # Логирование
    └── backup.py              # Автобэкапы
```

## 🚀 Запуск

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Создай `.env` файл (скопируй `.env.example`):
```
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_telegram_id
```

3. Запусти бота:
```bash
python main.py
```

## 🔧 Админ-панель

Команда: `/admin` (только для ADMIN_ID)

- 📊 Статистика бота (пользователи, поиски, активность)
- ➕ Добавить группу стёкол
- 🗑 Удалить группу стёкол
- 📩 Рассылка всем пользователям
- 👥 Список пользователей
- 🔥 Популярные запросы
- 💾 Бэкапы (создание + просмотр)
- ⚙️ Настройки (партнёрские ссылки)

## 🌍 Языки

Бот автоматически определяет язык пользователя:
- 🇷🇺 Русский (ru)
- 🇬🇧 English (en)  
- 🇸🇦 العربية (ar)

## 🔗 Партнёрские ссылки

После подбора стекла бот показывает ссылку на магазин.
Настроить ссылки можно в `config.py` → `PARTNER_LINKS`

## 📊 База данных

Бот автоматически создаёт `bot.db` с таблицами:
- `users` — пользователи
- `search_history` — история поисков
- `popular_searches` — популярные запросы
- `settings` — настройки бота
- `broadcasts` — рассылки

## 🛡 Безопасность

- `.env` и `*.db` не попадают в Git
- Автобэкапы при запуске
- Логирование всех ошибок
