## ШАГ 8: Запусти бота

```bash

```

Скрипт сделает:

- ✅ Обновит код (git pull)
- ✅ Установит зависимости

- ✅ Запустит нового бота
- ✅ Проверит что работает

**Должно вывести:**

## 📋 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### Бот не запускается

```bash
# Проверь логи
tail -f logs/bot.log

# Проверь ошибки
tail -f logs/errors.log

# Проверь .env файл
cat .env

# Запусти вручную
python3 main.py
```

### Ошибка "ModuleNotFoundError"

```bash
pip3 install python-telegram-bot==20.7
pip3 install python-dotenv==1.0.0
```

### Ошибка "TOKEN" или "BOT_TOKEN"

Проверь `.env`:
```bash
cat .env
```

Должно быть точно:
```
BOT_TOKEN=7972473497:AAG15HNZMxMOcIVxRD2Vj8si2khPQaTjgZA
ADMIN_ID=5164389862
```

Если неправильно — пересоздай:
```bash
rm .env
nano .env
# вставь правильные данные
```

### Бот запустился но не отвечает в Telegram

1. Проверь что токен правильный
2. Проверь что бот не запущен дважды:
```bash
ps aux | grep python
pkill -f "python.*main.py"
bash deploy.sh
```
