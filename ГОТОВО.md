2. Введите модель: `redmi 10`
3. Бот покажет группу совместимости

### Поиск АКБ:
1. Нажмите **🔋 АКБ**
2. Введите модель: `Samsung A54`
3. Бот покажет аккумулятор с ценой в BYN

**Варианты поиска АКБ:**
- `iPhone 15` → точный поиск
- `аккумулятор Samsung` → русское ключевое слово
- `АКБ iPhone` → русское ключевое слово
- `батарея Xiaomi` → русское ключевое слово
- `battery Google` → английское ключевое слово
- `Samsung A54` → по модели
- `BP47` → по артикулу
